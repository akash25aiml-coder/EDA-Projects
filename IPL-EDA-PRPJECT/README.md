#  IPL Data Analysis (EDA)

##  Project Overview

This project performs **Exploratory Data Analysis (EDA)** on IPL ball-by-ball dataset to uncover insights about teams, players, and match strategies, this data from 2005-2025.

---

##  Objectives

* Clean and preprocess real-world cricket data
* Analyze team and player performance
* Understand match-winning strategies
* Generate actionable insights using data visualization

---

##  Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

##  Key Analysis

### Team Analysis

* Teams with most wins

### Player Analysis

* Top run scorers
* Top wicket takers
* Strike rate analysis

### Match Strategy

* Batting first vs chasing analysis

### Venue Analysis

* Match distribution across venues

---

##  Key Insights

* Certain teams consistently dominate IPL
* Top players contribute significantly to match outcomes
* Chasing provides slight advantage
* Player efficiency (strike rate) is crucial
* Multiple factors influence match results
